Project Overview

This project examines how emotional intent can be inferred from short-form text in cases where conventional sentiment cues are sparse, ambiguous, or shaped by informal language. A particular emphasis is placed on emojis as semantic amplifiers rather than decorative noise.

Instead of treating emojis as isolated tokens or filtering them out entirely, the system incorporates emoji-aware feature engineering into a structured sentiment analysis workflow. Each stage—from data ingestion to model inference—is designed to remain transparent, reproducible, and grounded in exploratory analysis.

{# Project overview: narrative context before technical pipeline #}

System Overview

This project explores sentiment inference in low-resource, short-form social media text, where meaning is frequently compressed into symbolic cues. In these contexts, emojis function as high-signal affective amplifiers that can reinforce, soften, or redefine textual intent.

Rather than treating emojis as noise, the system implements auxiliary feature engineering. By extracting symbolic polarity and joining it with TF-IDF textual representations, the pipeline enables the classifier to evaluate linguistic and symbolic sentiment channels simultaneously.

The workflow encompasses the full data science lifecycle: deterministic cleaning, exploratory signal analysis, feature fusion, and interpretability diagnostics. Each stage produces immutable artifacts—from validated datasets to serialized modeling states—ensuring transparency and parity between training and inference.
